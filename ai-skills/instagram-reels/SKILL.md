## 🧠 Entendiendo el Algoritmo de Instagram (2025-2026)

### Señales de Ranking — Prioridad Alta
1. **Tiempo de reproducción completa** — el Reel que termina = empujado por el algoritmo
2. **Guardados** — señal de valor, supera al like en peso
3. **Compartidos a stories y DM** — distribución externa = calidad premium
4. **Comentarios reales** — no solo emojis
5. **Reproducciones repetidas** — el mejor indicador de contenido valioso

### Lo Que Penaliza
- Subtítulos/captions con links externos (usá el link en bio)
- Marcas de agua de TikTok visibles
- Baja resolución de video
- Música no disponible en la región

## 🎬 Sistema de Reels — Estructura Interna

### La Regla de los 3 Segundos
Los primeros 3 segundos determinan si el usuario sigue mirando o scrollea. Siempre abrís con:
- Una afirmación polémica o sorprendente
- Una pregunta que genera curiosidad
- Una promesa de valor inmediata
- Una acción visual impactante

### Estructura de Reel de 15–30 segundos (Óptimo)
```
[0-3 seg] HOOK — Para el scroll
[3-8 seg] SETUP — Contexto mínimo
[8-20 seg] CONTENIDO — La promesa cumplida
[20-25 seg] GIRO/SORPRESA — El momento "wow"
[25-30 seg] CTA — Qué hacer ahora
```

### Estructura de Reel de 30–60 segundos (Educativo/Story)
```
[0-3 seg]   HOOK provocador
[3-10 seg]  "Te cuento algo que no sabías sobre X"
[10-30 seg] 3-5 puntos de valor numerados
[30-50 seg] Aplicación práctica / ejemplo real
[50-60 seg] CTA + micro-gancho para el próximo
```

### Estructura de Reel de 60–90 segundos (Deep Dive)
```
[0-3 seg]   Hook + promesa grande
[3-15 seg]  Historia/contexto que engancha
[15-45 seg] Desarrollo con datos o ejemplos
[45-70 seg] El "secreto" o la revelación
[70-85 seg] Aplicación práctica para el viewer
[85-90 seg] CTA + "Guardalo para cuando lo necesités"
```

## 📸 Posts de Feed — Formatos y Captions

### Carrusel (máximo engagement del feed)
**Slide 1:** Hook visual (el que para el scroll)
**Slides 2-8:** Desarrollo del contenido — una idea por slide
**Slide último:** CTA + "Guardá este post"

**Fórmula de caption para carrusel:**
```
[Primera línea = Hook — sin cortar con "más"]
[Línea vacía]
[2-3 líneas de valor]
[Línea vacía]
Deslizá para ver todo 👉
[Línea vacía]
[CTA]
[Línea vacía]
[Hashtags — 5-10 relevantes y de nicho]
```

### Post de Producto Solo
```
[Descripción sensorial del producto — qué se siente, cómo luce]
[Precio / disponibilidad]
[CTA con fricción baja]

📦 Envíos a todo el país
💬 Consultanos por DM
🔗 Link en bio para comprar

#TiendaTuti #[categoría] #ModaArgentina #[temporada]
```

### Post de Lifestyle / Inspiración
```
[Frase corta + impactante — puede ser una pregunta]

[Descripción del look o momento]

¿Vos qué elegís para [ocasión]? Comentá 👇

#Lifestyle #TiendaTuti #OOTD #ModaAR
```

## 🎵 Música y Tendencias

### Estrategia de Música para Reels
- Usá tracks que estén en tendencia (aparecen con flecha ↗️)
- Para productos: música vibrante, sin letra que distraiga
- Para lookbooks: beats estilizados, lo-fi hip hop, trap suave
- Para fútbol: himnos, cumbias, música que genera emoción colectiva
- Para sorteos: música de expectativa y celebración

### Sincronización con el Beat
Alinear los cortes de video con el beat de la música aumenta hasta 40% el tiempo de reproducción.
