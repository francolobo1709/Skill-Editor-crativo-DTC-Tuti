## 🎯 Tipos de Flyer y Sus Reglas

### 1. Flyer de Oferta / Liquidación
**Objetivo:** Urgencia + conversión inmediata

**Estructura obligatoria:**
```
[ZONA 1 - HEADLINE: 40% del espacio]
  → Máximo 3-4 palabras, tipografía enorme
  → Ejemplo: "SALE 🔥", "50% OFF", "LIQUIDACIÓN TOTAL"

[ZONA 2 - PRODUCTO: 35% del espacio]
  → Imagen del producto hero, bien iluminado
  → Precio tachado + precio nuevo en color contrastante

[ZONA 3 - DETALLES + CTA: 25% del espacio]
  → Condiciones en letra más pequeña
  → CTA claro: "Comprá ahora", "Ir a la tienda", "Link en bio"
  → Logo de la marca
```

**Paleta:** Negro/rojo/blanco o naranja/negro/blanco
**Tipografía:** Bebas Neue o Anton (títulos) + Poppins (detalles)

---

### 2. Flyer de Sorteo / Giveaway
**Objetivo:** Participación + nuevos seguidores

**Estructura obligatoria:**
```
[ZONA 1 - Premio VISUAL: 45% del espacio]
  → Foto del premio, bien presentada, con brillo o aura
  → Valor del premio destacado en badge circular

[ZONA 2 - TÍTULO: 20% del espacio]
  → "🎁 SORTEAMOS" o "¡GRAN SORTEO!" muy grande
  → Fecha del sorteo

[ZONA 3 - MECÁNICA: 35% del espacio]
  → Pasos numerados para participar (máximo 3)
  → CTA final: "¡Participá ya!"
  → Logo + Instagram handle
```

**Paleta:** Morado/dorado/blanco o rosado/blanco/dorado
**Tipografía:** Impact, Bebas Neue o Black Han Sans

---

### 3. Flyer de Evento / Lanzamiento
**Objetivo:** Awareness + anticipación

**Estructura obligatoria:**
```
[ZONA 1 - FECHA Y NOMBRE: 30% del espacio]
  → Fecha enorme como elemento de diseño
  → Nombre del evento en tipografía premium

[ZONA 2 - VISUAL CENTRAL: 45% del espacio]
  → Imagen conceptual, producto, o elemento simbólico
  → Puede ser abstracto/artístico

[ZONA 3 - INFO + CTA: 25% del espacio]
  → Lugar, horario, link
  → Logo + redes sociales
```

**Paleta:** Según identidad de marca, colores nocturnos para eventos
**Tipografía:** Serif moderna (Playfair Display) o sans geométrica

---

### 4. Flyer de Fútbol / Mundial
**Objetivo:** Engagement emocional + viralización

**Estructura:**
```
[FONDO: Campo de fútbol o estadio con blur + overlay color nacional]

[ZONA 1 - SELECCIÓN/EQUIPO: 40%]
  → Camiseta o jugador icónico
  → Escudo con brillo/efecto premium

[ZONA 2 - DATO O MENSAJE: 35%]
  → "¡VAMOS ARGENTINA!" / "¿Quién gana hoy?"
  → Estadísticas atractivas o predicciones

[ZONA 3 - MARCA + MECÁNICA: 25%]
  → Si hay sorteo: cómo participar
  → Logo TiendaTuti integrado con elegancia
```

**Paleta:** Celeste/blanco/dorado (Argentina) o colores del equipo
**Tipografía:** Bebas Neue + Oswald

---

### 5. Flyer de Nueva Colección / Producto
**Objetivo:** Deseo + consideración de compra

**Estructura:**
```
[FONDO: Textura premium, degradado suave, color de temporada]

[ZONA 1 - PRODUCTO HERO: 55%]
  → Fotografía profesional del producto
  → Iluminación dramática lateral

[ZONA 2 - NOMBRE Y DESCRIPCIÓN: 25%]
  → Nombre de la colección o producto
  → 1-2 características clave

[ZONA 3 - PRECIO Y CTA: 20%]
  → Precio con styling premium
  → "Disponible ahora" / "Nueva temporada"
  → Logo
```

## ✍️ Sistema de Copy para Flyers

### Fórmula AIDA aplicada a flyers visuales
- **A** (Atención): Headline enorme — 2-5 palabras, tipografía de impacto
- **I** (Interés): Propuesta de valor — ¿qué ganás? ¿cuánto ahorrás?
- **D** (Deseo): Imagen del producto/premio que crea FOMO
- **A** (Acción): CTA directo — verbo en imperativo + beneficio

### Templates de Headline según tipo
| Tipo | Template | Ejemplo |
|------|----------|---------|
| Descuento | [%] OFF + [categoría] | "40% OFF Calzado" |
| Sorteo | SORTEAMOS + [premio] | "SORTEAMOS Zapatillas Nike" |
| Tiempo | Solo [tiempo] | "Solo hasta el domingo" |
| Cantidad | Últimas [N] unidades | "Últimas 10 unidades" |
| Lanzamiento | NUEVO + [producto] | "NUEVO Invierno 2026" |
| Evento | [Acción] + [fecha] | "El Mundial arranca el 11 de Junio" |
